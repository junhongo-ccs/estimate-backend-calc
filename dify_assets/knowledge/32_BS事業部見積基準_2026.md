# BS本部共通 見積・損益管理基準 (FY2026)

本ドキュメントは、株式会社NTTデータCCS BS本部におけるプロジェクト見積および収益性管理の標準基準を定めたものである。AI見積もりシステムの計算ロジックおよびLLMによる解説の根拠として使用する。

## 1. 労務費算定基準 (Labor Cost Standards)

労務費は、1人月あたり**160時間**（1日8時間 × 20日）を基準として算定する。

### 職位別月額コスト (Rank Costs)
利益管理表に基づき、以下のランク別単価を適用する。

| ランク | 対応職位 (PDF等) | 月額コスト (円) | 時間単価 (円/h) |
| :--- | :--- | :--- | :--- |
| **Rank 4** | L4相当 | 1,098,000 | 6,860 |
| **Rank 3** | L3相当 | 944,000 | 5,900 |
| **Rank 2** | L2相当 | 758,000 | 4,740 |
| **Rank 1** | L1相当 | 541,000 | 3,380 |

### 標準チーム構成
特に指定がない場合、以下の比率で混合したチーム構成を標準とする。
*   **Rank 3**: 80%
*   **Rank 2**: 20%

## 2. 間接費 (Indirect Costs)

間接費は率による一括計算ではなく、**所属部門別の「間接費単金（円/時）」**に総稼働時間を乗じて算出する。

*   **算出式**: `総工数(人日) × 8時間 × 部門別間接費単金`
*   **各部門の単金例**:
    *   ビジネスイノベーション事業部共通: 2,340円
    *   ＳＦ＆Ｍ事業部共通: 2,030円
    *   ＣＳシステム事業部共通: 1,940円
    *   ＤＴ事業部共通: 2,320円
    *   ソリューションビジネス推進室: 3,570円

## 3. 販管費 (SG&A Expenses)

販管費は**売上総利益（粗利）**に対して計上する。

*   **販管費率**: `本部販管費率(部門別) + 全社販管費率(一律 47.8%)`
*   **適用対象**: 売上高ではなく、**粗利（売上高 - 売上原価）**に対して乗算する。
*   **全社販管費率**: BS事業部においては **47.8%** で固定。

## 4. 応援プロジェクト (Supporting Project)

複数の部門が関与するプロジェクトでは、各部門の工数配分に基づき、間接費単金および本部販管費率を**加重平均**して適用する。これにより、実績と見込みの計数乖離を最小化する。

## 5. 収益性指標と逆算売価

### 指標定義
*   **売上原価 (COGS)** = 直接労務費 + 間接費 + 外注費 + 付随費用
*   **売上総利益 (Gross Profit)** = 売上高 - 売上原価
*   **営業利益 (Operating Profit)** = 売上総利益 - (売上総利益 × 販管費率)

### 逆算売価の計算 (Reverse Calculation)
目標とする営業利益率（τ）を達成するために必要な売価（Sales）は、以下の算式を用いる。
*   `Sales = COGS / (1 - τ / (1 - α))`
    *   α = 合計販管費率（本部＋全社）
